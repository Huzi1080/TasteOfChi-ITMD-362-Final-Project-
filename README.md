# Taste of Chi — Local Food Finder  

**Taste of Chi** is a modern, community-driven web platform designed to help users discover authentic, affordable, and locally loved restaurants, cafés, and hidden gems across Chicago. The goal is to make local dining exploration simple, inviting, and accessible for everyone — from residents to first-time visitors.

---

## Features

- **Restaurant Discovery** — Find curated local food spots filtered by neighborhood and price range.  
- **Clean, Responsive Design** — Fully functional layout optimized for desktop and mobile devices.  
- **Interactive Navigation** — Easy-to-use menu system (desktop + mobile toggle) for seamless browsing.  
- **Community Focused** — Highlights authentic and affordable food locations recommended by locals.  
- **Visual Gallery** — Showcases Chicago’s diverse food scene through curated images and user submissions.  
- **Contact & FAQ Pages** — Simple communication and user support integration.
